# docker-tet

## commands used

- install "rust"
    - >curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
- adding "rust" to path
    - >source "$HOME/.cargo/env"
- initialize mdbook project
    - >mdbook init

### usefull commands

- update book build
    - >mdbook build
- cleans up build output
    - >mdbook clean

### required dependencies

- build-essentials
